This Online Student Feedback System is a management information system for education establishments to manage student data. A Online Student Feedback System is an automatic feedback generation system that provides the proper feedback to the teachers as per the categories like always, poor, usually, very often, sometimes xisting system students can give feedback about the lecturers by doing manually. By this process student can give feedback in online system without wasting his time in writing. .
